Here are some programs and tools that could help you in your daily work.
It ranges from a simple command line tool to a full-featured desktop application.

## Command Line Tools

- [iTerm2](https://www.iterm2.com/)
- [Hyper](https://hyper.is/)
- [zsh](https://www.zsh.org/) in combination with [oh-my-zsh](https://github.com/robbyrussell/oh-my-zsh) lets your install plugins and themes.
  - [git shortcuts](https://kapeli.com/cheat_sheets/Oh-My-Zsh_Git.docset/Contents/Resources/Documents/index)
  - [zsh-autosuggestions](https://github.com/zsh-users/zsh-autosuggestions)
  - [fast-syntax-highlighting](https://github.com/zdharma/fast-syntax-highlighting)
- [fig](https://fig.io/) - adds VSCode-style autocomplete to your existing terminal
- [yarn](https://yarnpkg.com/)
- [nvm](https://github.com/creationix/nvm) - Node Version Manager. It's the easy way to manage multiple versions of Node.js.
- [starship](https://github.com/starship/starship)
- [homebrew](https://brew.sh/)

## Web -and Desktopapps

- [CodePen](https://codepen.io/) - is a web app for creating and sharing online short snippets of HTML, CSS, and JavaScript.

- [Sharefull](https://apps.apple.com/at/app/shareful/id1522267256?l=en&mt=12) - macOS Share Menu extension
- [Magnet](https://apps.apple.com/at/app/magnet/id441258766?l=en&mt=12) - macOS Window manager
- [Maccy](https://maccy.app/) - Clipboard manager for macOS
- [Roy Color Picker](https://www.useroy.com/) - a simple, powerful, and beautiful macOS color picker
- [Spline](https://spline.design/) - Easily create and publish 3D web experiences
- [Postman](https://www.getpostman.com/) - a powerful REST client
- [CleanMyMac X](https://macpaw.com/cleanmymac) - a powerful macOS system cleanup tool
- [Figma](https://www.figma.com/) - a powerful design tool
- [Pitch](https://pitch.com/) - the presentation tool for designers
- [PopSQL](https://popsql.com/) - a nicer way to write and run SQL queries

## Figma Plugins

I recommend to explore the community files of other creators. They often have a lot of useful components like a [Social Media Kit](https://www.figma.com/community/file/929473783107528634/Social-media-kit).

- [Stark](https://www.figma.com/community/plugin/732603254453395948/Stark) - The suite of integrated accessibility tools for your product design and development team
- [Figmotion](https://www.figma.com/community/plugin/733025261168520714/Figmotion) - Figmotion is an animation tool built right in Figma
- [Unsplash](https://www.figma.com/community/plugin/738454987945972471/Unsplash) - Unsplash is a free, hi-res, high-quality photos for your design projects
- [Border](https://www.figma.com/community/plugin/740014625507871586/Border) - Border is a Figma plugin that lets you add borders to your layers
- [Font Scale](https://www.figma.com/community/plugin/741231992144144738/Font-Scale) - is a quick tool to generate a harmonious and consistent typography hierarchy.
- [SkewDat](https://www.figma.com/community/plugin/741472919529947576/SkewDat) - Live skew any shape and fine-tune with simple non-destrucive controls.
- [Smart Data](https://www.figma.com/community/plugin/751874861800404329/Smart-Data) - Smart data inserts data from a JSON file or random data according to the layer name
- [Arc](https://www.figma.com/community/plugin/762070688792833472/arc) - Curve to your text upward, downward or into a circle with ease
- [SmoothShadow](https://www.figma.com/community/plugin/788830704169694737/SmoothShadow) - Make a smooth shadow with easing curves.
- [Mesh Gradient](https://www.figma.com/community/plugin/958202093377483021/Mesh-Gradient) - Mesh Gradient generates a gradient image with an underlying 2d mesh, and lets you edit the vertices & edges of the mesh to get the perfect gradient look.
